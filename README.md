Created as a personal project as part of the [boot.dev](http://www.boot.dev) curriculum.

A simple turn-based text RPG python project run from the command line.

If using a bash terminal, after cloning the source code the project can be run with
```
./main.sh
```
from the project root, or
```
$PATH_TO_PROJECT_ROOT/main.sh
```
This project is a learning exercise and is not being actively maintained.
